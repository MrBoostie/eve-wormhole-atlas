# Starter Expedition (Tier 1 → Tier 2)

## Objectives
1. Survive and extract consistently.
2. Build repeatable mapping/rolling discipline.
3. Generate ISK via C3 sites + salvage + opportunistic hunting.

## Initial Staging
- Prefer C2 with HS/C3 or C2 with LS/C3 depending risk appetite.
- Seed at least 2 scanner alts inside from day one.

## 72-Hour Checklist
- Day 1: mapping discipline + safe bookmarks + static profiling.
- Day 2: controlled PvE windows with pickets.
- Day 3: content roll + hunter posture + extraction drill.

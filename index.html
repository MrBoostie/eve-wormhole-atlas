Operating doctrine

This is written for small crews living out of the chain temporarily or semi-mobile. It prioritizes survivability, repeatability, and fast decision-making over expensive hero fits.

1

Map before motion

Resolve all signatures, bookmark actual wormhole objects, log destination, life, mass, and time before moving value.

2

Haul with Viators

Use blockade runners for routine logistics. Split expensive cargo and require eyes on both sides of the hole.

3

Run controlled windows

PvE, gas, salvage, and hauling happen only with pickets, current scans, and a shared abort call.

4

Extract deliberately

Keep exit bookmarks, probes, recovery ISK, and safe-log options ready before the chain deteriorates.

Hard rule: if the map is stale, the operation pauses. If a new signature appears, treat it as hostile access until resolved.

Operational details

Daily field SOP

Run this in order. Short, boring, survivable. The void charges extra for improvisation.

Opening scan

  1. Check all bookmarks against current signatures.
  2. Resolve every new sig before value moves.
  3. Record wormhole type, destination, life, mass, and timestamp.
  4. Identify K162s and active traffic before declaring anything safe.

Movement discipline

  1. Scout both sides before each Viator trip.
  2. Split expensive loads into multiple trips.
  3. Use cloak-MWD correctly and do not autopilot through hope.
  4. Pause hauling when the map changes.

PvE window

  1. Pickets on active statics and high-value connections.
  2. Combat ships aligned or near escape bookmarks.
  3. Salvage after combat pressure is understood.
  4. Abort on new sig, hostile probes, unknown ships, or lost picket contact.

Extraction

  1. Stage exit bookmarks and safe logs.
  2. Keep probes on independent characters.
  3. Move irreplaceable cargo first, not last.
  4. Leave enough ISK outside the hole to recover.
Polarization reminder: weapon timers do not stop wormhole jumps, but polarization does. Track direction and timestamps like your hull depends on it, because yes, astonishingly, it does.

# EVE Wormhole Atlas

Private planning repo for wormhole expeditions.

## Scope
- Wormhole living playbooks
- Role-based ship fits
- Copy-paste multibuy shopping lists
- Character role plans (small/medium/alt-heavy)

## Structure
- `docs/` concepts and references
- `fits/` ship fits by role
- `multibuy/` shopping lists for market import
- `ops/` SOPs and checklists

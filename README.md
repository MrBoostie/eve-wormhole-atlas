# EVE Wormhole Atlas

A practical wormhole operations handbook and GitHub Pages site for small-gang / alt-heavy EVE Online wormhole expeditions.

Live site: <https://mrboostie.github.io/eve-wormhole-atlas/>

## Scope

- Wormhole living and mobile expedition playbooks
- Daily chain mapping, picket, PvE, salvage, and extraction SOPs
- Role-based ship fits and character role plans
- Viator-first logistics doctrine
- Copy-paste multibuy shopping lists
- Clean public landing page for navigation and field reference

## Structure

- `index.html` public GitHub Pages landing page
- `assets/` site CSS/JS
- `docs/` concepts, plans, and references
- `fits/` ship fits by role
- `multibuy/` market import lists
- `ops/` SOPs and checklists

## Editing

The site is intentionally static: edit `index.html`, `assets/site.css`, and `assets/site.js`, then push to `main`. GitHub Pages serves the root of the repository.

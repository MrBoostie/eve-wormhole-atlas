# EVE Wormhole Atlas

A practical GitHub Pages field handbook for small-gang wormhole expeditions: Viator-first logistics, Astero-first scouting, copy-paste fits, multibuy packs, and daily SOPs.

Live site: https://mrboostie.github.io/eve-wormhole-atlas/

## Doctrine bias

- Viators for routine logistics.
- Astero before Stratios for early scouting/hunting.
- Temporary/mobile staging before structures.
- No blind hauling, stale maps, or PvE without pickets.

## Key pages

- `index.html` — polished single-page site
- `fits/` — copy-paste ship fits
- `multibuy/` — exact EVE multibuy lists
- `ops/` — operational SOPs and checklists

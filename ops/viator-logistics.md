# Viator Logistics SOP

## Core Rules
- Never move blind. Scout every gate/hole first.
- Use instadock/instaundock bookmarks.
- Split high-value cargo across runs.

## Execution Pattern
1. Scout route with separate pilot.
2. Confirm pre-align bookmarks.
3. Cloak+MWD timing on risky transitions.
4. If chain changes, pause and rescan.
5. On threat spike, abort route and safe-log.
